# AX Discovery Portal - Web App

Next.js 기반 웹 애플리케이션 (Phase 2)

## 기능 (예정)

- Signal Inbox (생성/필터/태깅/배정)
- Scorecard 입력/조회
- Brief 생성/승인
- Play Dashboard
- KPI Digest 뷰

## Tech Stack

- Next.js 14+
- React 18+
- TypeScript
- Tailwind CSS
- shadcn/ui

## Quick Start

```bash
cd app/web
npm install
npm run dev
```
