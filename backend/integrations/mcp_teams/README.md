# Teams MCP Server

Microsoft Teams 연동을 위한 MCP 서버 (Phase 2)

## 기능 (예정)

- Webhook을 통한 알림 전송
- Adaptive Card 지원
- 승인 요청/응답 처리

## 환경 변수

```
TEAMS_WEBHOOK_URL=https://outlook.office.com/webhook/xxxxx
```

## Tools (예정)

- `teams.send_message` - 메시지 전송
- `teams.send_card` - Adaptive Card 전송
- `teams.request_approval` - 승인 요청
